# Task 1 - Create AnalysisService

Implement:

app/Services/AnalysisService.php


Responsibilities:

- Analyze conversation content using OpenAIService.
- Use PromptService for AI prompts.
- Extract structured information from customer conversations.

The analysis should include:

- intent
- sentiment
- priority
- customer status
- required action
- summary
- whether human intervention is required


AI response must use JSON format.

Example:

{
    "intent": "refund_request",
    "sentiment": "negative",
    "priority": "high",
    "customer_status": "existing_customer",
    "requires_human": true,
    "summary": "Customer requested refund",
    "recommended_action": "Explain refund policy"
}


Requirements:

- Create database storage if needed.
- Create model if needed.
- Add relationships.
- Use proper Laravel conventions.
- Add validation/casting.
